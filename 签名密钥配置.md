## 📝 配置 GitHub Secrets

### 步骤 1: 打开 GitHub Secrets 设置

访问：https://github.com/jieke5235/match-download/settings/secrets/actions

### 步骤 2: 添加私钥 Secret

1. 点击 **New repository secret**
2. Name: `TAURI_PRIVATE_KEY`
3. Secret: 复制上面的私钥内容（整个字符串）
4. 点击 **Add secret**

### 步骤 3: 添加密码 Secret（可选）

由于生成密钥时没有设置密码，你可以：

**选项 A**: 不添加密码 Secret
- GitHub Actions 会自动处理空密码的情况

**选项 B**: 添加空密码 Secret
1. 点击 **New repository secret**
2. Name: `TAURI_KEY_PASSWORD`
3. Secret: 留空
4. 点击 **Add secret**
